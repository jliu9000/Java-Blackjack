CSE 361: Java Blackjack Project
Authors: Jordan Brown, Matthew Keller, John Liu

The following contains the source code to the Java Blackjack Project.
The program will launch from the GUI class, and uses mainly the GUI and Game class to interact with other class within the game.
Because this is a Java Applet intended to be embedded in a web browser, it will not run as a stand alone jar.
Easiest way to run this program is to compile and run is in eclipse as a Java Applet with GUI is the starting class.
